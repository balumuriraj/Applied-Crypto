------------------------------ReadMe----------------------------
1. On extracting the folder, it consists of the Java file and Certs folder that contains the certificates.
2. Open command prompt and goto this folder location and execute the command "javac CertOps.java".
3. The Java file is compiled and a class file will be generated in the same folder.
4. Now to execute the Class file, enter the command "java CertOps".
5. The output is shown on the command prompt.
